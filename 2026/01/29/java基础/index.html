# 五 集合  
1. 集合框架体系：  
集合主要是两组（单列集合、双列集合）  
Collection 接口有两个重要子接口 List、Set，他们的实现子类都是单列集合  
Map 接口的实现子列是双列集合，存放 k-v
**单列集合**    
Collection：List、Set  
List：ArrayList、Vector、LinkedList  
Set：HashSet、TreeSet  
**双列集合**  
Map：HashMap、HashTable、TreeMap   
HashTable：Properties  
HashMap:LinkedHashMap
